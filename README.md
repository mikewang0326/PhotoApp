# PhotoApp

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

PhotoApp is Pixabay photo display App.

  - Display different types of photoes
  - View image details
  
### Tech

PhotoApp uses a number of open source projects to work properly:

* [Language] - Kotlin
* [Architectural Pattern] - Android Architecture Components
* [HTTP] - Retrofit
* [UI] - RecyclerView
* [Image] - Glide
* [Data] - https://pixabay.com/api/
